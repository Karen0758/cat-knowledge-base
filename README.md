# Cat Parent Knowledge Base

An English cat-care knowledge base built from a local StashBase `cat` space.

This repository collects feline care guidelines, veterinary reference PDFs, and a reader-friendly HTML handbook for new cat parents. The goal is to make the source material easier to browse by care task: feeding, home setup, medical prevention, mature and senior care, behavior, pain, urinary disease, FIP, and zoonotic risk.

## Start Here

- Open [`index.html`](./index.html) for the handbook-style guide.
- Browse [`index/`](./index/) for the original PDF references.
- If you use StashBase, this repository also keeps the extracted Markdown and asset folders generated from the PDFs so the knowledge base can remain searchable after cloning.

## What Is Included

- A single English HTML handbook: [`index.html`](./index.html)
- Original PDF reference files under [`index/`](./index/)
- StashBase-generated hidden Markdown extraction files, used for search and retrieval
- StashBase metadata files: [`STASHBASE.md`](./STASHBASE.md) and `.stashbase/`

The Chinese HTML draft from the local workspace is intentionally not included.

## Knowledge Areas

The handbook organizes the material around practical caregiver questions:

- Food selection, treat limits, feeding behavior, and diet transitions
- Litter box sizing, indoor enrichment, and multi-cat tension
- Vaccination, cat-friendly veterinary visits, oral health, and anesthesia
- Mature and senior cat monitoring, pain, muscle condition, and quality of life
- Urinary stones, therapeutic diets, FIP diagnosis, NSAID use, and zoonotic risk

## Medical Disclaimer

This repository is for learning and care preparation only. It does not replace veterinary diagnosis or treatment. Urinary blockage, breathing difficulty, seizures, poisoning, persistent vomiting, trauma, prolonged anorexia, or sudden severe behavior changes should be handled by a veterinarian or emergency clinic.
